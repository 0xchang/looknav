# LookNav

一款为 Typecho 设计的简约网址导航主题，采用清新浅绿色风格，支持分类展示、站点收藏、多引擎搜索、农历时间等特性。

---

## 功能特性

- **分类导航** — 左侧分类目录，一键锚点跳转
- **站点收藏** — 浏览器 LocalStorage 本地收藏，独立"我的收藏"视图
- **多引擎搜索** — 支持百度、Bing、Google 一键切换搜索
- **实时时间** — 右上角显示实时时间，悬停查看农历、干支、生肖、节气、节日
- **自定义图标** — 每篇文章可单独设置图标地址，或自动获取站点 favicon
- **响应式布局** — 完美适配桌面端与移动端
- **毛玻璃 UI** — 现代轻量设计风格，支持自定义背景图

---

## 安装方法

1. 下载本主题文件
2. 将主题文件夹上传至 Typecho 的 `/usr/themes/` 目录
3. 确保文件夹名为 `looknav`
4. 登录后台 → 控制台 → 外观 → 启用 LookNav

## 使用说明

### 1. 添加导航内容

本主题的核心逻辑：

| 概念 | 对应 |
|------|------|
| **分类** | 导航分组（如：开发工具、常用网站） |
| **文章** | 导航条目（如：GitHub、百度） |

**操作步骤：**

1. 后台 → 管理 → **分类**，创建导航分组
2. 后台 → 撰写 → **撰写文章**：
   - **标题**：网站名称（如 `GitHub`）
   - **分类**：选择对应分组
   - **正文**：填写网站简介（显示在卡片下方）
   - **自定义字段** → **导航链接**：填写外部网址（如 `https://github.com`）
   - **自定义字段** → **自定义图标**：填写图标 URL（留空则自动获取 favicon）
   - 发布文章

### 2. 主题设置

后台 → 控制台 → 外观 → **设置外观**

| 设置项 | 说明 |
|--------|------|
| 站点 LOGO | 网站 Logo 图片地址 |
| Favicon API | 第三方图标服务地址，如 `https://favicon.im/`。留空则直接读取站点 `/favicon.ico` |
| ICP 备案号 | 页面底部备案信息 |
| 萌 ICP 备 | 页面底部萌备案信息 |
| 背景图地址 | 支持完整 URL 或本地路径（如 `/usr/themes/looknav/bg.jpg`），留空使用默认渐变 |

### 3. 收藏功能

- 点击卡片右侧的 **☆**（空心星）收藏站点
- 点击 **★**（实心橙星）取消收藏
- 收藏的站点保存在浏览器本地，换设备不互通
- 点击左侧 **"我的收藏"** 可独立查看所有收藏

## 注意事项

1. **收藏功能** 基于浏览器 `LocalStorage`，清除浏览器数据会丢失收藏
2. **农历时间** 支持 1900-2100 年，悬停右上角时间即可查看
3. **自定义图标** 优先级高于自动获取的 favicon
4. 移动端左侧分类目录会自动隐藏，点击右上角菜单按钮（如有）或直接滑动浏览
